Etiquetas HTML



Â¿Que son las etiquetas?


Las etiquetas HTML son partes de codigo con funciones especializadas para ciertas tareas, algunas para hacer secciones, otras para crear campos de texto, algunas decorativas, etc.

hr

La etiqueta hr se utiliza para crear lineas horizontales en tu pagina, se pueden usar para resaltar un texto o separar secciones


Div

La etiqueta Div se usa como una especie de caja, en la cual puedes almacenar otros elementos como titulos o textos como este mismo cuadro blanco


Input

Input es una etiqueta usada en html para recibir datos del usuario, cualquiera, es una etiqueta que se usa sobretodo en formularios para rellenar su informacion o enviarlo

Rellena este input y dale a el boton BUSCAR


br

La etiqueta br se utiliza para crear un salto de linea, separando un texto de otro o agregar mas espacio entre dos secciones


Footer

La etiqueta footer es una seccion del texto que se usa normalmente abajo de toda la pagina, para agregar las redes sociales, creditos, o algunas aclaraciones, abajo de aqui hay un footer, que es el ejemplo.
